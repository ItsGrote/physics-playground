# Physics Playground

C++ project using SFML to simulate physics concepts in real time.

## 📌 Current status
- Basic project structure created
- CMake configured
- CLI menu implemented (WIP)

## 🧪 Simulations planned
- MRU
- MRUV
- Projectile motion
- Pendulum
- Spring-mass system
- Gravitation
- Collision

## 🚀 How to Run

```bash
mkdir build
cd build
cmake ..
make
./sim
```
